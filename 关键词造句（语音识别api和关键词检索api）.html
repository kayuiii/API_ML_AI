关键词造句

该功能运用到语音识别API和关键词检索API


AI部分：


1.用户点击【按住说话】，录入语音资料。


2.语音识别API：

①输入用户录入的语音资料；

②进行语音识别；

③返回识别结果。


3.关键词检索API：

①输入用户录入的语音资料以及题目要求的关键词；

②进行关键词检索；

③返回识别结果。


4.识别后分为：用户录入语音资料中含有关键词、用户录入语音资料中并未含有关键词两种情况。


不同情况的显示请看下图。


交互部分：


1.点击左上方【复习】图标即可退出评测，返回复习界面。


录入语音中含有关键词


1.根据语音识别API返回的结果，显示用户录入语音资料的文本内容。


2.根据关键词检索API返回的结果，对上述文本内容中的相应内容进行加粗标识。


3.在右下方呈现【下一题】按钮，点击即可跳转至下一题目。


4.点击左上方【复习】图标即可退出评测，返回复习界面。

录入语音中未含有关键词


1.根据语音识别API返回的结果，显示用户录入语音资料的文本内容。


2.在右下方呈现【好像有哪里不对哦，再来一次！】按钮，点击即可跳转回这道题目的测试界面。


3.点击左上方【复习】图标即可退出评测，返回复习界面。
